docsnap
=======

Docsnap is a webapp to make PDFs out of markdown docs in Dropbox. That is all.

http://yannesposito.com/Scratch/en/blog/Yesod-tutorial-for-newbies/

Requirements
============

```bash
cabal update
cabal install cabal-dev
cabal-dev install snap -fhint
```
